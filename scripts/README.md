# Scripts

Utility scripts for development, deployment, and maintenance.

## Available Scripts

- `analyze_backend.py` - Analyze backend code quality
- `analyze_frontend.py` - Analyze frontend code quality
- `validate_quality.py` - Validate code quality standards
- `deploy.sh` - Deploy to production
- `setup_dev.sh` - Set up development environment
- `run_tests.sh` - Run all tests
- `backup_db.sh` - Backup database

## Usage

Make scripts executable:
```bash
chmod +x scripts/*.sh
```

Run a script:
```bash
./scripts/setup_dev.sh
```
