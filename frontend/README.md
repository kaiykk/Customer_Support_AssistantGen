# Frontend - AssistGen

Vue 3 + TypeScript frontend application for the AssistGen intelligent customer service system.

## Structure

- `src/` - Source code
  - `components/` - Vue components
  - `views/` - Page views
  - `stores/` - Pinia stores
  - `services/` - API service clients
  - `assets/` - Static assets
- `public/` - Public static files
